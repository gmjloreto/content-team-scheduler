# 📸✨ Media Team Scheduler — GMJ Loreto

Bem-vindo ao **Media Team Scheduler** da GMJ Loreto! 🚀
Um sistema simples, rápido e eficiente para organizar a **escala da equipe de mídias** durante eventos importantes como o Tríduo Pascal ✝️🔥

---

## 🎯 Objetivo

Facilitar a organização da equipe responsável por:

* 📸 Fotografia
* 🎬 Edição
* 📱 Storymaking

Tudo isso em um só lugar, de forma visual, colaborativa e em tempo real.

---

## 🧠 Como funciona

🔹 Cada dia possui seus eventos (missas, celebrações, etc.)
🔹 Ao clicar em um evento, você pode:

* Inserir seu nome
* Escolher sua função
* Confirmar sua participação

🔹 O sistema automaticamente:

* Limita vagas por função
* Atualiza em tempo real
* Mostra quem já está escalado

---

## ⚙️ Regras da Escala

| Função        | Limite    |
| ------------- | --------- |
| 🎬 Editor     | 1 pessoa  |
| 📸 Fotografia | 2 pessoas |
| 📱 Storymaker | 1 pessoa   |

✔ Evita sobrecarga
✔ Garante organização
✔ Distribui melhor a equipe

---

## ⚡ Funcionalidades

* ✅ Cadastro rápido (sem login)
* 🔄 Atualização em tempo real
* ❌ Remoção com confirmação
* 📱 Interface otimizada para celular
* 🎨 Layout simples e moderno
* ⚙️ Integração com banco de dados (Supabase)

---

## 📦 Tecnologias utilizadas

* HTML
* CSS
* JavaScript
* Supabase (banco de dados + realtime)

---

## 🚀 Como usar

1. Acesse o link abaixo 🌐
2. Escolha o dia e evento
3. Clique no evento desejado
4. Insira seu nome
5. Escolha sua função
6. Clique em **Confirmar**

Pronto! Você já está escalado ✅
